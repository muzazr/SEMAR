# Smart Efficient Medical Record
A project that functions as a real-time health condition record integration system covering heart rate, blood sugar, cholesterol, and uric acid levels based on Internet of Things (IoT).

# Requirements
## Hardware
* NodeMCU ESP8266
* Pulse Sensor
* Jumper cables
* Breadboard
* USB cable
## Software
* Arduino IDE
* Board Package: ESP8266
* Library: Blynk, ESP8266WiFi, WiFiClient
